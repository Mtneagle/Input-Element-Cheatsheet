< <span><input></span> Values Cheatsheet

The <input> element does not need a closing tag

The type=" " attribute can have the following values.

Note: "required" can be used as an attribute for any type value

Value Description Attributes Used With
checkbox Single checkbox item name=" " (use same name for group of checkboxes), value=" "
number Creates input field for numbers name=" ", min, max
password Creates single row input field with censored text name = " "
radio Creates a single radio button name=" " (use same name for group of buttons), value=" "
range Creates a slider between min and max value name=" ", min, max, step
submit A submit button with text "Submit" will be rendered value=" " to change text
text Creates single row input field for text name = " ", maxlength=" ", pattern=RegEx, minlength, maxlength


Other Elements used on Forms

These Elements have closing tags


Element Description Attributes Used With
<datalist> Basic search autocomplete function paired with <input> element <input> "list" value must match datalist ID. Several <option>s populate datalist
<form> Used to contain various input elements for submission method="post" action=" location to send data"
<label> Creates a label for an <input> element for=" " must equal <input> element ID
<select> Creates dropdown list name=" ", Choices in list from <option> elements
<textarea> Creates multi-row input field for text name = " ", rows, cols (set initial size)
